- nodejs / npm / bower / grunt-cli

## Install
```bash
# ensure nodejs, ruby, git etc are installed. also npm, bower, grunt, bundle, rubygems
git clone https://github.com/packadic/theme
cd theme
npm install
bower install
bundle install
```
